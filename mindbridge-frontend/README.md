# MindBridge Frontend

## Setup Instructions
1. Open `index.html` in your browser.
2. Each module is separated into HTML, CSS, and JS files.
3. Mock data is available in `/data`.

## Modules
- Chatbot
- Screening (PHQ-9 & GAD-7)
- Booking
- Resources Hub
- Forum
- Dashboard (Analytics)
